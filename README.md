# Harmon_Sign

Веб-приложение для визуального представления гармонического сигнала, его спектра, чистого спектра и отфильтрованного спектра.

## Возможности

- добавление нескольких гармоник в суммарный сигнал;
- вывод до 5 отдельных сигналов на общий график без добавления их в сумму;
- редактирование амплитуды, частоты, фазы и номера гармоники;
- включение и отключение гармоник без удаления;
- отображение суммарного сигнала, спектра, чистого спектра и спектра после низкочастотной фильтрации;
- горизонтальная промотка, масштабирование ширины графиков и логарифмическая шкала частоты для спектров;
- хранение гармоник в SQLite;
- экспорт текущих гармоник в JSON.

## Запуск

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
```

После запуска откройте `http://127.0.0.1:5000`.
